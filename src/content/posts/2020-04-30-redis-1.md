REmote DIctionary Server(Redis) 是一个由Salvatore Sanfilippo写的key-value存储系统。

Redis是一个开源的使用ANSI C语言编写、遵守BSD协议、支持网络、可基于内存亦可持久化的日志型、Key-Value数据库，并提供多种语言的API。

它通常被称为数据结构服务器，因为值（value）可以是 字符串(String), 哈希(Hash), 列表(list), 集合(sets) 和 有序集合(sorted sets)等类型。

**Redis的速度很快，因为**

-   redis完全基于内存，内存的读写速度非常快。
-   数据结构简单，对数据操作也简单。
-   使用多路复用I/O模型，多个网络连接复用同一个线程，可以让单个线程高效处理多个请求，避免了上下文切换的时间和性能损耗。

# Redis五种数据类型

## String（字符串）

String类型是二进制安全的，可以包含任何数据。

```
存入数据：SET test:count 1

取出数据：GET test:count

自增：incr test:count

自减：decr test:count
```

以上实例中，使用了redis的set、get、incr和decr命令，键为test:count，值为1。一个键最大能储存512M数据。

## Hash（哈希）

Hash类型的数据是一个键值对集合。是一个string类型的field和value的映射表，特别适合存储对象。

```
hset test:user id 1
"OK"
hset test:user username zhangsan 
"OK"
hget test:user id
"1"
hget test:user username
"zhangsan"
```

## List（列表）

Redis列表是简单的字符串列表，按照插入顺序排序，可以添加或取出一个元素从列表的头部或者尾部。列表有序可重复，可实现队列和栈的结构。

```
lpush test:ids 101 102 103 104 // 从列表左侧加入元素
(Integer) 4
llen test:ids   // 列表的长度
(Integer) 4
lindex test:ids 0    // 索引0位置上的value
"101"
lrange test:ids 0 2  // 索引0-2所有的value
1) "101"
2) "102"
3) "103"
rpop test:ids   //从列表右侧弹出一个value
```

## Set（集合）

Set是string的无序集合，不可重复。集合是通过哈希表实现的，添加、删除、查找的事件复杂度都为O(1)。

```
sadd test:teachers aaa bbb ccc ddd  // 向集合中增加元素
(Integer) 4
scard test:teachers   // 查看集合中元素的个数
(Integer) 4
spop test:teachers   // 从集合中随机弹出一个元素
bbb
smembers test:teachers   // 查看集合中所有的元素
1) "ccc"
2) "ddd"
3) "aaa"
```

## SortedSet（有序集合）

Zset和set一样，也是string类型元素的集合，且不允许重复。但是zset每个元素都会关联一个double类型的分数。redis通过这个分数为集合中的元素从小到大进行排序。分数可以重复。

```
zadd test:teachers 10 aaa 20 bbb 30 ccc 40 ddd  // 向集合中增加元素
(Integer) 4
zcard test:teachers   // 查看集合中元素的个数
(Integer) 4
zscore test:teachers ccc   // 查看集合中某元素的分数
"30"
zrank test:teachers ccc   // 查看集合中某元素分数的排名
(Integer) 2
zrange test:teachers 0 2   // 查看集合中0-2排名的所有元素
1) "aaa"
2) "bbb"
3) "ccc"
```
